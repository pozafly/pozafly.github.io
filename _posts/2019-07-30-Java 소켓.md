## 소켓이란?

### <네트워크>

서비스를 제공하는 입장에 있는 컴퓨터 : 서버(네이버)

서비스를 요청하는 입장에 있는 컴퓨터 : 클라이언트(웹브라우저)

 

[www.naver.com](http://www.naver.com) 을 치면

- DNS(해당 도메인의 ip주소를 얻는다)

- 게이트웨이(비트 연산자로 해당 ip 주소가 내부 네트워크인지 외부 네트워크인지 판별)

- OSI7계층(7계층을 통과하면서 데이터가 가공됨)

클라이언트가 서버에 접속하기 위해 알아야하는 정보는? DNS(도메인 네임 시스템). ip주소, port번호(프로그램 마다 번호가 부여되어있음).

 

#### ip주소를 알려면 ipconfig /all

- ipv4 주소 : 사설 ip라고 한다. 돈을 주고 안샀다.

- 기본 게이트 웨이 : wifi 공유기. 자동으로 내부적으로 ip주소를 생성해낸다. 충돌나지 않도록.

- 고정 ip : 고정으로 해두면 충돌이 날 수 있음. 그래서 자동으로 ip를 주게 만들어놨다. 0~255번 까지 가능하다. 넘어가면 부하가 일어남.

- DNS서버 : 각 세계 모든 도메인 네임이 다 있다. 구글, 통신사에서 제공. ip주소들을 제공해준다. 

- DNS서버 -> 게이트 웨이. 이 게이트웨이가 외,내부 판단.

### 서버

ServerSocket ss = new ServerSocket(5001); 

//괄호 안에 포트번호가 들어간다. 운영체제 보통 20개정도 프로그램 돌아가고 있다. 앞자리는 보통 예약된 번호이므로 번호 4자리로 해주는 것이 좋음.

System.out.println(s); //찍으면 소켓 주소가 나옴.

### 클라이언트

Socket s = new Socket("192.168.0.1", 5001);

- Socket의 생성자를 보자 : Socket(String host, int port)

- host : ip주소("www.naver.com" 혹은 "192.168.0.1" 혹은 "localhost")

- port : 서버측의 포트번호

```java
============================================
서버 : 
============================================
import java.net.*;
class Server{
	public static void main(String args[]){
		try(
			ServerSocket ss = new ServerSocket(5001);
			Socket s = ss.accept();
		){
			System.out.println(s);   //접속된 클라이언트 정보를 알 수 있다.
		}catch(Exception e){
			System.out.println(e);
		}
	}
}

============================================
클라이언트 : 
============================================
import java.net.*;
class Client{
	public static void main(String args[]){
		try(
			Socket s = new Socket("localhost", 5001);
		){
			System.out.println(s);
		}catch(Exception e){
			System.out.println(e);
		}
	}
}
```

[ex] 안녕~ 날리는 소스

```java
============================================
서버 : 
============================================
import java.net.*;
import java.io.*;
class Server{
	public static void main(String args[]){
		try(
			ServerSocket ss = new ServerSocket(5001);
			Socket s = ss.accept();
			InputStream in = s.getInputStream();  //socket클래스 안에 있는 getInputStream() : 접속한 클라이언트와 통신하게 해줌. Stream을 만드는 객체를 얻어와서 InputStream형식의 in 변수에 넣는다.(소켓에서 InputStream 얻기라고 생각하자.)
		){
			byte[] buffer = new byte[5];  //5byte가 날라올 것이니까 바이트 방 5개를 준비.
			in.read(buffer);
			System.out.println(new String(buffer));
		}catch(Exception e){
			System.out.println(e);
		}
	}
}

============================================
클라이언트 : 
============================================
import java.net.*;
import java.io.*;
class Client{
	public static void main(String args[]){
		try(
			Socket s = new Socket("localhost", 5001);
			OutputStream out = s.getOutputStream();  //소켓에서 OutputStream 얻기
		){
			String msg = "안녕~";  //총 5byte 이다. --> 안 : 2byte, 녕 : 2byte, ~ : 1byte
			out.write(msg.getBytes());
		}catch(Exception e){
			System.out.println(e);
		}
	}
}
```

여기서 단점은 보내는 byte를 알아야한다. 따라서 Scanner를 사용.

[ex] Scanner를 사용하여 cmd에 사용자가 입력한 문자를 보내보자.

```java
============================================
서버 : 
============================================
import java.net.*;
import java.io.*;
import java.util.*;           //Scanner는 java.util.* 안에 있음.
class Server{
	public static void main(String args[]){
		try(
			ServerSocket ss = new ServerSocket(5001);
			Socket s = ss.accept();
			InputStream in = s.getInputStream();
			Scanner sc = new Scanner(in);        //이쪽과
		){
			System.out.println(sc.nextLine());   //이쪽만 추가해주면 끝이다.
		}catch(Exception e){
			System.out.println(e);
		}
	}
}
============================================
클라이언트 : 
============================================
import java.net.*;
import java.io.*;
import java.util.*;
class Client{
	public static void main(String args[]){
		try(
			Socket s = new Socket("localhost", 5001);
			OutputStream out = s.getOutputStream();
			Scanner sc = new Scanner(System.in);
		){
			String msg = sc.nextLine() + "\n"; //Scanner에서 읽어들이려면 	반드시 역슬래시 n을 붙여줘야함. 한개의 라인이기 때문에.
			out.write(msg.getBytes());
		}catch(Exception e){
			System.out.println(e);
		}
	}
}
```

[ex] while문으로 계속해서 문자를 받는 소스 + stop문자를 보내면 연결 끊어짐.

```java
============================================
서버 : 
============================================
import java.net.*;
import java.io.*;
import java.util.*;
class Server{
	public static void main(String args[]){
		try(
			ServerSocket ss = new ServerSocket(5001);
			Socket s = ss.accept();
			InputStream in = s.getInputStream();
			Scanner sc = new Scanner(in);
		){
			while(true){                            //루프
				System.out.println(sc.nextLine());
				if(sc.nextLine().equals("stop")) break;        //만약 sc 스캐너로 받아온 문자가 stop이면 멈춰라
			}
		}catch(Exception e){
			System.out.println(e);
		}
	}
}
============================================
클라이언트 : 
============================================
import java.net.*;
import java.io.*;
import java.util.*;
class Client{
	public static void main(String args[]){
		try(
			Socket s = new Socket("localhost", 5001);
			OutputStream out = s.getOutputStream();
			Scanner sc = new Scanner(System.in);
		){
			while(true){
				String msg = sc.nextLine() + "\n"; 
				out.write(msg.getBytes());
				if(msg.equals("stop\n")) break;    //만약 sc 스캐너로 받아온 문자가 stop이면 멈춰라
			}
		}catch(Exception e){
			System.out.println(e);
		}
	}
}
```
