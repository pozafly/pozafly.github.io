## Enum 이란?

* 자기 자신의 값을 상수로 선언한 `class`

```java
class A{
	static final A B = new A();
	static final A C = new A();
	static final A D = new A();
}

위 아래는 같은 소스다.

enum A{
	B,C,D
}
```

따라서 A안에 없는 상수는 A. (주소를 말하는 것임) 으로 불러올 때 컴파일 에러가 발생.

<br/>

<br/>



### switch case문에 enum사용할 수 있다.

```java
switch(enum 변수){
	case enum의 값 : ;break;
}
```

<br/>

### values() : values는 enum으로 선언된 이름 형식의 배열로, 선언된 상수들을 모두 가져온다.

```java
[ex]
enum Gender{
	F,M
}
class A{
	public static void main(String args[]){
		Gender[] array = Gender.values();
		for(Gender g : array)
			System.out.println(g);
	}
}
```

<br/>

### enum 내부클래스

```java
[ex]
enum A{
	B,C
}
class D{
	enum AA{
		BB,CC
	}
	A o = A.B;
}
class E{
	D.AA o = D.AA.BB;
}
```

<br/>

### enum을 class처럼 사용가능. 생성자.

```java
[ex]
class ColorTable{
	enum Color{
		Red("빨강"), Green("초록"), Blue("파랑");
		
		String c;
		Color(String c){
			this.c = c;
		}
		public String toString(){
			return c;
		}
	}
	public static void main(String args[]){
		Color[] colorArr = Color.values();
		for(Color c : colorArr)
			System.out.println(c);
	}
}
```



- enum 예제

```java
enum MyColor {PINK,GRAY,GREEN}
class ColorTable {
    MyColor color;
    void setColor(MyColor color) {
        this.color = color;
    }
    String getColor() {
        String str = "";
        switch (color) {
            case PINK:str = "핑크";break;
            case GRAY:str = "그레이";break;
            case GREEN:str = "그린";
        }
        return str;
    }
}
class A {
    public static void main(String args[]) {
        ColorTable table = new ColorTable();
        // table.setColor(MyColor.RED); 
        table.setColor(MyColor.PINK);
        System.out.println(table.getColor());  //핑크
    }
}
```
