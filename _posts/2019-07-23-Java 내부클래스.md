## 내부 클래스란?

<br/>

>내부 클래스란, Java의 클래스 내부에 선언한 클래스를 말하는데, 내부 클래스를 사용하면 두 클래스의 멤버들을 서로 쉽게 접근할 수 있다는 장점과 외부에는 불필요한 관계 클래스를 감춤으로써 코드의 복잡성을 줄일 수 있다.
>
> 
>
>중첩 인터페이스도 가능. -> 주로 UI프로그래밍에 이용됨. 

<br/>

※ 종류

- `내부멤버클래스` : 클래스 안에 선언된 클래스

- `내부로컬클래스` : 클래스 안에 있는 메서드 안에 선언된 클래스.

멤버클래스와 로컬클래스는 위치에 따라 구분된다.



```java
[ex]
class A{
	class B{}  //내부멤버클래스 A$B.class
	void abc(){
		class C{}  //내부로컬클래스 A$1.class
	}
	void def(){
		class C{}  //A$2.class
		class D{}  //A$1D.class
	}
}

[ex]
class A {
    static class B {  //static 멤버변수를 가지고 있으므로 static으로 선언됨.
        static int num1 = 100;
    }
    class C {
        int num2 = 200; //그냥 내부멤버클래스.
    }
}
class Test {
    public static void main(String args[]) {
        System.out.println(A.B.num1);   //A 안에 있는  B라는 static class의 static 멤버변수
        A o1 = new A();
        A.C o2 = o1.new C();    //단순 내부클래스에 접근할 때는 A형태와  A주소가 필요하다.   
        System.out.println(o2.num2);
    }
}
```



- `내부무명클래스` : 대부분 기존클래스의 오버라이딩이나 추상클래스의 추상메소드 구현, 인터페이스의 추상메소드 구현을 목적으로 한다.

```java
[sy]
new 부모클래스(){
	오버라이딩 메소드
};


[ex]
class A{
	void abc(){System.out.println(1);}
}
class B{
	public static void main(String args[]){
		A o = new A(){  //즉 o는 다형적 변수. A형태이지만, 내부무명클래스(자식) 주소를 생성했다.
			//여기는 클래스안에 들어가는 로직. 즉 이 클래스는 선언되지만, 이름이 없다.
			void abc(){System.out.println(2);}   //A클래스 메소드를 오버라이딩 함.
		};
		o.abc();   //따라서 오버라이딩 된 자식 메소드를 호출함.
	}
}
```
